
# Arogya Mithra

**Arogya Mithra** is a healthcare web application developed using microservice architecture to provide secure, scalable, and efficient digital healthcare services. The platform enables patient registration, health record management, appointment scheduling, and automated notifications through independently managed services, ensuring better maintainability and reliability.

## Tech Stack

* **Backend:** FastAPI (Python)
* **Database:** PostgreSQL
* **Messaging:** RabbitMQ
* **Caching:** Redis
* **API Gateway:** Traefik / Nginx
* **Containerization:** Docker, Docker Compose
* **Authentication:** JWT
